# PRD2018-G07  
  

1.要求：  
    （1）建整合分支，小组长分配任务，整合人员在integration分支发布相关任务模板；  
    （2）小组每人自建分支，首先从integration分支上拉取任务相关信息，完成分配的任务后各自交到自己分支上；  
    （3）最后由整合人员负责整合并提交到integration分支。   
	注：小组成员（非组长）只保留自己的分支。如果需要查看其它成员分支，只能在本地新建分支拉取查看，不能推送。
